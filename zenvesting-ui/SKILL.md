# Zenvesting UI: The Visual Architect

## Aesthetic Standards
- **Theme:** Dark Mode (#0e1117 background).
- **Accents:** Gold (#d4af37) for buttons, headers, and highlights.
- **Tone:** Premium, sleek, and "Zen".

## Components
- **Host Control:** Clean layout for question navigation and game reset.
- **Player View:** Distraction-free interface with focused question containers.
- **Leaderboard:** Elegant tables with gold headers.

## Interaction Design
- Use `st.markdown` with custom CSS for elements that standard Streamlit components can't reach.
- Implement "Zara's Engagement" features: floating emojis, fire animations, and interactive feedback.
